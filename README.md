# Binary-v.s-Sequential-Search-
The Java program "Problem 5.1" is about comparing Sequential and Binary Search. Generates random arrays, sorts them with QuickSort, and measures index results and comparison counts across multiple sizes. Includes a loop that tests array sizes (16, 64, 256, 1024, 4096, 16384) and prints a formatted performance table. Showing the each of the searches efficiency.
